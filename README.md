# Customer & Product Intelligence Platform

## Business Problem

A beauty retailer wants to improve:
- Customer retention
- Product performance
- Revenue growth
- Marketing effectiveness

## Objectives

1. Identify high-value customers
2. Measure customer retention
3. Understand product purchasing behaviour
4. Detect churn risk
5. Deliver executive dashboards

## Technology Stack

- SQL
- Snowflake
- dbt
- Tableau
- GitHub

## Expected Outcomes

- Customer segmentation
- Cohort analysis
- Customer lifetime value
- Product affinity analysis
- Executive dashboards
